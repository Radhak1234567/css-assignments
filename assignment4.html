CSS Box Model Demonstration

Content-Box Model

In the content-box model, the width and height of the box are applied only to the content. Padding and borders increase the overall size of the element.

Box 1

Padding: 20px
Border: 5px

Box 2

Padding: 20px
Border: 5px

Border-Box Model

In the border-box model, the width and height of the box include the content, padding, and borders. The overall size of the element stays the same, regardless of padding and border size.
